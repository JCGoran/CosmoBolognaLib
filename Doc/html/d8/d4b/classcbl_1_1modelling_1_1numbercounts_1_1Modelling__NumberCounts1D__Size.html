set the cosmological parameters and parameters necessary to re-parametrise the void size function

the model has N cosmological parameters, the tracer bias and the coefficients (b_slope and b_offset) of the linear relation necessary to convert the tracer bias into the value of the bias computed inside cosmic voids (see Contarini et al. 2019)
