Detailed Description

The class Modelling_DensityProfile.

Modelling_DensityProfile.h "Headers/Modelling_DensityProfile.h"

This file defines the interface of the base class Modelling_DensityProfile, used for modelling cluster surface density profile measurements, i.e. \(\Delta\Sigma(r)\) [ \(h\) M \(_\odot\)/pc \(^2\)]. Cosmological units are forced.
