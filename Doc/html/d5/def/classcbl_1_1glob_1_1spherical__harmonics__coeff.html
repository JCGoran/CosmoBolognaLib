Detailed Description

The class spherical_harmonics_coeff.

This class is used to handle objects of type spherical_harmonics_coeff . It is used to perform the Slepian&Eisenstein algorithm to compute the three-point correlation function It contains all methods to compute spherical harmonics expansion \(a_{lm}\) of a collection of points in separation bins, and the product of \(a_{lm}\) for two separation bins.
