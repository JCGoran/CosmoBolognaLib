Detailed Description

The class TwoPointCorrelation1D_filtered.

TwoPointCorrelation1D_filtered.h "Headers/TwoPointCorrelation1D_filtered.h"

This class is used to handle objects of type TwoPointCorrelation1D_filtered . It is used to measure the filtered monopole of the two-point correlation function, \(w(r_c)=2\pi \int dr \xi(r) W(r,r_c) r^2\), where \(W(x) = (2x)^2(1-x)(0.5-x)r_c^{-3}\), and \(\xi(r)\) is the monopole of the two-point correlation function
